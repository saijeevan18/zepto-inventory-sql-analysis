# Zepto Inventory SQL Analysis

This project analyzes the Zepto inventory dataset using MySQL.

## Skills Used
- MySQL
- GROUP BY
- Aggregate Functions
- ORDER BY
- Data Analysis
- Power Bi

## Business Questions Solved
1. Categories with highest number of products
2. Products with highest discount percentage
3. Categories with highest average discount
4. Out-of-stock products
5. Categories with most out-of-stock items
6. Most expensive products
7. Categories with highest average price
8. Largest package size products
9. Categories with highest available stock
10. Products with low remaining stock

## Business Insights(Power Bi)
- Category-wise product distribution
- Discount analysis
- Expensive product identification
- Inventory availability analysis

## Dataset
Zepto Inventory Dataset from Kaggle.
